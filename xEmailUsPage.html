Main Navigation Keyboard Access

The main navigation can be accessed by using the tab key.

Once inside the navigation, the arrow keys are used to navigate up and down, and across the different menu levels.

To go down a menu level use the down arrow key when the drop down arrow icon is highlighted.

To go up a menu level, use the up arrow key when any menu item in the bottom levels are highlighted.

To go across the menu items, use the tab key or the left and right keys.
